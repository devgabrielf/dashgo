
## 💻 Sobre o projeto

Dashboard desenvolvido durante a trilha de ReactJS do bootcamp Ignite da Rocketseat, com o qual aprendi sobre a construção de interfaces modernas, responsivas e acessíveis com o Chakra UI.

---

## ⚙️ Tecnologias

- NextJS
- Chakra UI
- MirageJS
- ApexChartsJS
- ReactQuery
- Faker
